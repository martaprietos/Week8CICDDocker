<h2>Local (H2)</h2> 
<ul>
<li>To run the application locally, run Week7JPA in the dev profile and go to http://localhost:8080/h2-console.</li>
<li>Go to  http://localhost:8080/swagger-ui/index.html to see and use CRUD operations.</li>
<li>Add new person entries using POST, view entries with GET, and delete or update entries with DELETE and PUT.</li>
<li>To view entries in the H2 console, write "SELECT * FROM PERSONS" and click run.</li></ul>


<h2>Docker (Postgres)</h2>
<ul><li>To run the application using Postgres, run Week7JPA in the dev profile and run docker-compose, make sure Docker Desktop is running as well.<br></li></ul>
